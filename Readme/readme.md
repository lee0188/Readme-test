## 就是書要你 Book Hunting

A website to find book resources you need.

找到你所需的圖書資源
## Installation

Use the git clone

```bash
git clone https://github.com/lee0188/Finals
```

## Usage

```bash
#go to the first directory

cd Finals/first

#run django

python3 manage.py runserver

#click the development server

http://127.0.0.1:8000
```
